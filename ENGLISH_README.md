# Credit Risk Analytics Platform

An end-to-end credit risk modeling platform designed with production-grade architecture, MLOps practices, and business-driven financial metrics.

---

## 🧠 Business Context

A fintech institution needs to:

- Estimate Probability of Default (PD)
- Calculate Expected Loss (EL)
- Optimize approval strategies
- Reduce financial risk while maintaining profitability

This project simulates a real-world credit risk environment and builds a deployable scoring system using open-source tools.

---

## 🏗 Architecture Overview

Data Layer (SQL / Simulated Data)  
→ Feature Engineering  
→ Model Training  
→ MLflow Experiment Tracking  
→ Model Evaluation  
→ API Deployment (FastAPI)  
→ Production Simulation  

---

## ⚙️ Technology Stack (100% Open Source)

- Python
- Scikit-learn
- XGBoost
- MLflow (MLOps & experiment tracking)

- SQL (feature engineering layer)
- Git & Branching strategy

## 📊 Business Metrics
